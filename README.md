A super simple way to create UI from data. Oui maps data to a user interface. Pass it an object and Oui creates a set of controls. _O_bjects go in, _UI_ comes out

```javascript
oui({
  someNum : 10,
  dir: {
    someNestedProp: 10  
  }
})
```
